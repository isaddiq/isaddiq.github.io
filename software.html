ReUnixchange

Revolutionary data exchange platform for BIM interoperability and standardized information sharing across AEC workflows.

Data Exchange BIM Interoperability OBJ CSV XML Autodesk Revit Unity

Revolutionary Data Exchange Platform

ReUniXchange is a cutting-edge Revit plugin that enables seamless export of BIM geometry, metadata, and materials to Unity. It supports modular construction workflows and immersive XR applications by bridging the gap between design and visualization environments. The plugin allows users to assign shared parameters (ModularID) to Revit group components, extract structured metadata into CSV format, export 3D views as OBJ files with MTL and texture data, and re-import any changes made in Unity through an XML file.

For modular projects, the plugin can automatically generate ModularID parameters based on group names, enabling logical tracking and reconstruction of elements inside Unity. Metadata from Revit views can be exported to CSV and attached to 3D objects inside Unity to enable semantic enrichment, selection, and inspection during runtime. Material definitions used in Revit are also exported into Unity-readable MTL files with associated texture images.

Core Infrastructure Components

  • Modular ID Parameter Assignment in Revit
  • Semantic and Metadata Extraction to CSV
  • Material Export and Unity MTL Conversion
  • 3D Model Export with Textures and Materials
  • Unity Hierarchy Creation for Components
  • Bidirectional XML-based Data Exchange

BIM Network Graph Maker

Professional data visualization solutions for Building Information Modeling workflows. Transform your architectural and engineering data into interactive network diagrams.

BIM IFC Network Graph ArchiCAD Revit CSV Semantic relationships

Automatic Network Graph Generation

This specialized application is designed for AEC professionals who need to visualize complex relationships within their BIM models. Each tool processes CSV/IFC data exports (by ReUniXchange for Autodesk Revit and ArchUniXchange for ArchiCAD, and IFC export as typical IFC from either BIM authoring tool) to create interactive network diagrams that reveal connections, dependencies, and hierarchies in your building information models.

Core Features

  • Create Network Diagram for IFC, and CSV
  • Create Hierachy based on different metadata
  • Automatic Calculation of elements and its types
  • IFC semantic Verification
  • ArchiCAD and Autodesk Revit IFC and CSV support
