ReUnixchange

Revolutionary data exchange platform for BIM interoperability and standardized information sharing across AEC workflows.

Data Exchange BIM Interoperability OBJ CSV XML Autodesk Revit Unity

Revolutionary Data Exchange

ReUniXchange is a Revit plugin that enables seamless export of BIM geometry, metadata, and materials to Unity. It supports modular construction workflows and immersive XR applications by bridging the gap between design and visualization environments. The plugin allows users to assign shared parameters (ModularID) to Revit group components, extract structured metadata into CSV format, export 3D views as OBJ files with MTL and texture data, and re-import any changes made in Unity through an XML file.
For modular projects, the plugin can automatically generate ModularID parameters based on group names, enabling logical tracking and reconstruction of elements inside Unity. Metadata from Revit views can be exported to CSV and attached to 3D objects inside Unity to enable semantic enrichment, selection, and inspection during runtime. Material definitions used in Revit are also exported into Unity-readable MTL files with associated texture images.
The OBJ export process converts the current 3D view into geometry and materials that can be directly imported into Unity. To preserve material assignment, users must not rename the OBJ and MTL files. Material reassignment inside Unity can only be customized if Unity has already linked them automatically via the Legacy Material Import feature in the Inspector.
Changes made inside Unity, such as repositioning modules, adjusting metadata, or reconfiguring object hierarchy, can be exported from Unity into an XML file. The ReUniXchange plugin in Revit can then read this XML and apply the changes back into the original BIM model. This enables a complete bidirectional data exchange workflow from design to immersive visualization and back to BIM.
The plugin is ideal for architects, engineers, researchers, and developers working in modular construction, BIM-to-XR pipelines, and digital twin applications. It supports Autodesk Revit versions 2022 through 2025 and Unity versions 2021.3 or later. The plugin outputs files in CSV, OBJ, MTL, and XML formats and stores them locally without any external transmission.
ReUniXchange is part of a research initiative to integrate BIM and immersive environments through data-driven workflows and modular logic. It is designed to be lightweight, intuitive, and compatible withboth design and runtime systems.

Core Architecture

  • Modular ID shared Parameter Assignment in Autodesk Revit
  • Semantic and Metadata Extraction to CSV file
  • Autodesk material export and conversion to Unity mtl format
  • 3D Model export to OBJ from Autodesk Revit with material and texture
  • Semantic enrichment inside Unity and Hierarchy Creation for 3D Components
